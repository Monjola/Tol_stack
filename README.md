# StackMaster | Tolerance Analysis

A web-based tolerance analysis tool for engineering and manufacturing applications.

## Description

StackMaster is a single-page web application that helps engineers and designers perform tolerance stack-up analysis. The tool provides an intuitive interface for calculating dimensional tolerances and analyzing how component variations affect overall assembly dimensions.

## Features

- Interactive tolerance analysis interface
- Real-time calculations
- Modern, dark-themed UI
- Responsive design

## Usage

Simply open `index.html` in a web browser to use the application. No additional setup or installation required.

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
